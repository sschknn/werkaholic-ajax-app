# Workspace: c:\coding

This workspace contains the setup and configuration for the user's development environment.

## Workspace Structure

- **scripts/**: Contains utility scripts for installation and maintenance (e.g., Kilocode CLI installation).
- **.agent/workflows/**: Stores agentic workflow definitions.
- **.gemini/**: Agent memory and artifacts.

## Kilocode CLI Installation

To install the Kilocode CLI, run the appropriate script from the `scripts/` directory:

### Windows
```cmd
.\scripts\install_kilocode.bat
```

### Unix/Linux/macOS
```bash
./scripts/install_kilocode.sh
```

For more details on the Kilocode CLI, visit the [official documentation](https://github.com/kilocode/kilocode-cli).
